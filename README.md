# 休閒網頁遊戲平台 (Web Casual Games)

**開發者：林先生 (hklamsir) | 協作 AI：OpenClaw (小信 🦋)**

## 1. 專案概述
本專案是一個整合型的休閒網頁遊戲平台，提供「隨開隨玩、輕鬆療癒」的娛樂體驗。所有遊戲均採用現代網頁技術 (HTML5/JS) 開發，無需安裝，支援電腦與行動裝置瀏覽器。

**[🎮 立即遊玩 (Play Now)](https://hklamsir.github.io/web-casual-games/)**

## 2. 遊戲列表

### 🐱 1. 貓咪咖啡廳大亨 (Cat Café Tycoon)
- **類型：** 放置經營 (Idle / Clicker)
- **風格：** 溫馨可愛 SVG 畫風
- **玩法：**
    - 扮演店長，點擊招攬客人賺取金幣。
    - 解鎖各種可愛貓咪員工（提供自動收入）。
    - 升級店內設施，解鎖成就。
    - **特色：** 自動存檔功能 (LocalStorage)，隨時回來繼續經營。

### 🐍 2. 霓虹貪吃蛇 2077 (Neon Snake 2077)
- **類型：** 動作街機 (Arcade)
- **風格：** 賽博龐克 (Cyberpunk) 霓虹視覺
- **玩法：**
    - 經典貪吃蛇機制，配合現代化流暢手感。
    - **特殊技能：** 「相位衝刺」(Phase Dash) 可短暫穿牆與加速。
    - 適應性畫布設計，支援觸控滑動操作。

### 👾 3. 太空入侵者 (Space Invaders)
- **類型：** 復古射擊 (Retro Shooter)
- **風格：** 經典像素風格
- **玩法：**
    - 駕駛戰機，抵禦波狀攻擊的外星艦隊。
    - 左右移動躲避彈幕，精準射擊擊落敵機。
    - 挑戰神秘的 UFO 以獲取高額獎勵分數。
    - **特色：** 完整的音效體驗（射擊、爆炸、背景音樂），專為手機優化的虛擬按鍵操作。

### 🐶 4. 表情對對碰 (Emoji Memory Match)
- **類型：** 益智記憶 (Puzzle / Memory)
- **風格：** 清新粉嫩配色 (Pastel)
- **玩法：**
    - 翻開卡片，找出兩張相同的表情符號。
    - 考驗短顯記憶力，越快完成越好。
    - **特色：** 響應式卡片翻轉動畫，自動計時與計步。

### 🌌 5. 重力反轉 (Gravity Switch)
- **類型：** 跑酷 / 反應 (Runner / Arcade)
- **風格：** 極簡暗黑科技風 (Minimalist Sci-Fi)
- **玩法：**
    - 點擊螢幕切換重力方向（上/下），讓角色奔跑於天花板或地板。
    - 避開隨機生成的障礙物，堅持越久分數越高。
    - **特色：** 高速流暢的物理手感，粒子特效回饋。

### ⚛️ 6. 量子 2048 (Quantum 2048)
- **類型：** 益智滑塊 (Puzzle / Sliding Block)
- **風格：** 量子科技風 (Deep Space / Neon)
- **玩法：**
    - 滑動方塊，將相同數字的方塊合併升級。
    - 目標是合成出傳說中的「2048」態量子方塊。
    - **特色：** 流暢的滑動動畫，支援移動端手勢操作，記錄最高分。

### 🧱 7. Zen Stack (禪意堆疊)
- **類型：** 物理堆疊 (Physics / Stacking)
- **風格：** 極簡粉彩 (Minimalist Pastel)
- **玩法：**
    - 點擊螢幕讓左右移動的方塊落下。
    - 對齊下方方塊以保持最大面積，未對齊部分將被切除。
    - 挑戰最高堆疊層數，連續完美對齊可增加寬度或分數。
    - **特色：** 舒緩的視覺效果，簡單卻令人上癮的單指操作。

### 🎵 8. 音律流動 (Synth Flow)
- **類型：** 節奏音樂 (Rhythm / Music)
- **風格：** 賽博龐克 / Synthwave
- **玩法：**
    - 音符從上方落下，在到達判定線時點擊對應軌道。
    - 鍵盤使用 D、F、J、K 操作四個軌道，手機使用底部按鈕。
    - **Perfect** 時機獲得最高分數，**Good** 稍有偏差，**Miss** 會中斷連擊。
    - 累積連擊 (Combo) 可獲得額外加分！
    - **特色：** 程式自動生成節拍譜面，霓虹視覺效果，Web Audio API 音效回饋。

### ✍️ 9. 墨韻織詞 (Word Weaver)
- **類型：** 文字解謎 (Word Puzzle)
- **風格：** 東方水墨書法 (Ink Brush Aesthetic)
- **玩法：**
    - 在字母格子中滑動連接相鄰字母，組成隱藏的成語或詞語。
    - 找出所有目標詞語即可過關，隨關卡提升難度增加。
    - **特色：** 東方水墨美學設計，流暢的滑動連線動畫，成語數據庫，限時挑戰。

### 🏎️ 10. 霓虹漂移 (Neon Drift)
- **類型：** 一鍵街機 (One-tap Arcade / Drifting)
- **風格：** 賽博龐克 (Cyberpunk) 霓虹視覺
- **玩法：**
    - 點擊螢幕或空格鍵切換漂移方向。
    - 保持在路徑中央，不要撞到邊緣。
    - 收集紫色能量方塊獲取額外分數。
    - **特色：** 無限生成路徑，隨著分數提高速度加快，極簡操作與極限反應。

### 🔧 11. 電幻修復師 (Glitch Sorter)
- **類型：** 街機分類 (Arcade Sorting)
- **風格：** 賽博龐克 / 數據流 (Data Flow)
- **玩法：**
    - 拖動從上方落下的核心，將其對準左右對應的傳輸口。
    - **紅色核心** 拖向左側，**藍色核心** 拖向右側。
    - 能量條歸零則連線斷開遊戲結束。
    - **特色：** 快速反應挑戰，精美的霓虹發光特效，隨分數提升的動態難度。

### 💎 12. 棱鏡脈衝 (Prism Pulse)
- **類型：** 益智消除 (Match-3 / Collapse)
- **風格：** 霓虹光影 (Neon / Prism)
- **玩法：**
    - 點擊 3 個或以上相連的同色棱鏡進行消除。
    - 一次消除越多，分數越高並產生連鎖反應。
    - **特色：** 炫酷的棱鏡特效，流暢的下落動畫，提示系統，漸進式難度。

### 🎣 13. 霓虹釣客 (Neon Angler)
- **類型：** 時機反應 / 垂釣 (Timing / Arcade)
- **風格：** 賽博龐克 (Cyberpunk) 霓虹視覺
- **玩法：**
    - 點擊螢幕放下魚鉤，捕捉游動的數據魚。
    - 避開病毒核心，避免分數與能量損失。
    - **特色：** 獨特的垂釣手感，發光粒子效果，隨時間增加的能量消耗機制。

### 🦇 14. 回聲獵人 (Echo Chase)
- **類型：** 視覺回聲 / 感知反應 (Sensory Arcade)
- **風格：** 極簡暗黑 (Dark / Ripple)
- **玩法：**
    - 點擊螢幕發出回聲波，照亮黑暗中的目標。
    - 在波紋消失前快速點擊出現的金色光點。
    - **特色：** 獨特的「視覺定位」機制，極簡主義美學，動態難度系統（光點存活時間隨等級縮短）。

## 3. 技術棧 (Tech Stack)
- **核心：** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+)
- **繪圖：** HTML5 Canvas API (貪吃蛇、太空入侵者、霓虹漂移), SVG DOM 操作 (貓咪咖啡廳)
- **部署：** GitHub Pages
- **版本控制：** Git
